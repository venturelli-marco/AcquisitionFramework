# AcquisitionFramework
Framework for acquiring and saving images acquired from Kinect One and EXLs3(IMU sensor).

## Prerequisites
* [Python 2.7.x and higher](https://www.python.org/)  
* [NumPy](http://www.numpy.org/) 
* [OpenCV-Python](http://opencv.org/) 
* [PyKinect2](https://github.com/Kinect/PyKinect2)
* [PyQt4](https://wiki.python.org/moin/PyQt)
